# Unite Pick Vision

**ポケモンユナイトの敵チーム構成を自動で読み取り、OBSの配信画面にリアルタイム表示するツールです。**

ピック画面を見るだけで、敵5体のロール・CC・レーン予測がオーバーレイに自動表示されます。AIや高性能GPUは不要、完全オフラインで動作します。

![platform](https://img.shields.io/badge/platform-Windows%20%2F%20macOS-blue) ![license](https://img.shields.io/badge/license-MIT-green)

---

## こんな人におすすめ

- ポケモンユナイトをOBSで配信・録画している
- ピック画面で敵構成をメモするのが手間だと感じている
- 視聴者に敵チームの情報をわかりやすく見せたい

---

## 主な機能

| 機能 | 説明 |
|---|---|
| 敵チーム自動検出 | ピック画面を見てAI不要で5体を自動認識 |
| リアルタイム表示 | ピック中は右下にメモ帳風で随時更新 |
| 確定後ビッグ表示 | 5体確定後にロール別カラーカードで大きく表示 |
| レーン予測 | ロール・ジャングル適性から上/中央/下を自動推定 |
| CC情報表示 | スタン・スロー・ノックバックなどを色分け表示 |

---

## 動作環境

- **Windows 10 または Windows 11**（推奨）
- **OBS Studio 28以降**（無料）
- **キャプチャボード** でSwitchの映像をPCに取り込んでいること
- macOSはソースから起動可能（動作は確認済み）

## 導入手順

### ステップ1：ツールをダウンロードする

1. このページ上部の **「Releases」** をクリック（または[こちら](https://github.com/harisen/unite-pick-vision/releases)）
2. 最新バージョンの **`unite-pick-vision-win64.zip`** をクリックしてダウンロード
3. ダウンロードした `.zip` ファイルを**右クリック → 「すべて展開」** で好きな場所に展開

> 📁 展開先の例：`C:\Games\UnitePickVision\`（デスクトップでもOK）

---

### ステップ2：OBSのWebSocket機能を有効にする

Unite Pick Vision は OBS と通信して画面を読み取ります。まずOBS側で通信機能をオンにします。

1. **OBS Studio を起動**する
2. 上のメニューバーから **「ツール」→「WebSocket サーバー設定」** をクリック

   > 「WebSocket」が見当たらない場合：OBS のバージョンが古い可能性があります。[OBS公式サイト](https://obsproject.com/ja)から最新版をインストールしてください（28以降で標準搭載）

3. 以下のように設定する：

   | 項目 | 設定値 |
   |---|---|
   | WebSocket サーバーを有効にする | ✅ チェックを入れる |
   | サーバーポート | 4455（変更不要） |
   | 認証を有効にする | ✅ チェックを入れる（推奨） |

4. 「**認証を有効にする**」にチェックを入れた場合、画面に**パスワードが表示**されます。このパスワードをメモしておいてください（後で使います）

5. **「OK」** をクリックして閉じる

---

### ステップ3：OBSにゲーム画面を映す

Unite Pick Vision はOBSに映っているゲーム画面を読み取ります。まだキャプチャの設定をしていない場合は以下の手順で設定します。

1. OBS の **「ソース」** 欄の下にある **「＋」** をクリック
2. **「映像キャプチャデバイス」** を選択
3. デバイスの一覧から**キャプチャボードの名前**を選択
4. OBS のプレビュー画面に**Switchのゲーム画面が映っていること**を確認

> ⚠️ OBSのプレビューにゲーム画面が映っていないと、Unite Pick Vision は正常に動作しません。

---

### ステップ4：Unite Pick Vision を起動する

1. 展開したフォルダを開き、**`Unite Pick Vision.exe`** をダブルクリック

2. 以下のような警告が表示された場合：

   **「WindowsによってPCが保護されました」（SmartScreen）**
   → 「**詳細情報**」をクリック → 「**実行**」をクリック

   > この警告は有料のコード署名をしていないために出るものです。ソースコードはすべてこのページで公開しており安全です。

3. 起動すると、画面右下のタスクバーに**小さな丸いアイコン**が表示されます

   > 見つからない場合：タスクバー右端の **「∧」** をクリックすると隠れたアイコンが表示されます

---

### ステップ5：OBSのパスワードを設定する

1. タスクバーのアイコンを**右クリック** → **「設定」** をクリック
2. 設定画面が開くので、ステップ2でメモした **OBSのパスワード** を入力
3. **「保存」** をクリック

   > パスワードを設定しなかった場合（認証なし）は空欄のまま保存でOKです

4. 保存後、OBS のソース欄に **`obs-vision overlay`** が自動で追加されます

---

### ステップ6：オーバーレイの表示順を確認する

OBS のソース欄で、**`obs-vision overlay` がゲームキャプチャより上（前面）にある**ことを確認してください。

```
ソース欄（上が前面）
┌─────────────────────────┐
│ obs-vision overlay  ← ここが上にある必要がある
│ 映像キャプチャデバイス
└─────────────────────────┘
```

下にある場合はドラッグして上に移動してください。

---

### 設定完了！使い方

1. OBS を起動してゲーム画面を映す
2. `Unite Pick Vision.exe` を起動
3. ポケモンユナイトの**ピック画面（選択フェイズ）になると自動で検出開始**
4. 敵のピックが進むにつれて右下のメモ帳に随時表示されます
5. 敵5体が確定すると、ロール別カラーカードが大きく表示されます
6. **60秒後に自動で非表示**になり、次の試合を待ちます

> 2回目以降はOBSを起動した状態でツールを起動するだけでOKです。設定は保存されています。

## うまく動かない場合

### アイコンが表示されない / 接続できない
- OBSが起動しているか確認してください
- OBSのWebSocket設定（ステップ2）が正しく設定されているか確認してください
- トレイアイコンを右クリック → **「OBSに再接続」** を試してください

### 検出されない / 精度が悪い
- OBSのプレビューにピック画面がはっきり映っているか確認してください
- キャプチャボードの解像度が低い場合、検出精度が下がることがあります（1280×720以上推奨）
- スロット座標のズレが原因の場合は、下記の「スロット座標の調整」を試してください

### SmartScreenで実行できない
- 「詳細情報」→「実行」をクリックしてください
- それでも実行できない場合はWindowsのセキュリティ設定を確認してください

## スロット座標の調整（上級者向け）

画面レイアウトが標準と異なる場合は座標を調整できます。

1. OBSにピック画面を映した状態で `http://localhost:38921/capture` をブラウザで開く（スクショ保存）
2. `http://localhost:38921/calibrate.html` を開く
3. 表示された画面で、1番上のアイコンの**左上→右下**、1番下のアイコンの**中央**の順にクリック
4. **「この座標を適用」** で保存

---

## テスト動画での検証（開発者向け）

実際に配信しなくてもオフラインでテストできます。

1. ピック画面を含む動画を録画しておく
2. ツールフォルダ内の `.env` ファイルをメモ帳で開き、以下を追記：
   ```
   TEST_VIDEOS=C:/path/to/video1.mp4;C:/path/to/video2.mp4
   ```
3. トレイアイコン右クリック → **「▶ テスト動画で検証」**

## ソースから起動（開発者向け）

```bash
git clone https://github.com/harisen/unite-pick-vision.git
cd unite-pick-vision/obs-vision
npm install
cp .env.example .env  # .envを編集してOBS設定を入力
npm start
```
